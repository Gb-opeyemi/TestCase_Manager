# TestCase Manager

TestCase Manager is a role-based web application for managing software test cases.

## Project structure

```text
.
|-- public/
|   |-- css/
|   |   `-- styles.css
|   |-- js/
|   |   `-- main.js
|   `-- index.html
|-- src/
|   |-- config/
|   |   `-- database.js
|   `-- server.js
`-- package.json
```


## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
